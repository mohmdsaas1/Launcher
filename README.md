# Simple Launcher — MVP

## حالة الكود
هذا الكود **ما تم بناؤه ولا تشغيله فعليًا** — البيئة اللي كُتب فيها ما فيها Android SDK ولا اتصال بمستودعات جوجل (dl.google.com). يعني احتمال وجود خطأ صغير (نسخة مكتبة، تعارض Gradle) وارد ولازم تتأكد بنفسك أول ما تفتحه.

## كيف تشغّله (بلابتوب — Android Studio)
1. افتح المجلد في Android Studio (Open an existing project).
2. خلي Gradle يعمل Sync (يحتاج إنترنت أول مرة عشان يحمّل المكتبات).
3. وصّل جهاز حقيقي أو شغّل Emulator (Android 8.0 / API 26 فأعلى).
4. اضغط Run.
5. أول ما يفتح التطبيق بيوديك لإعدادات "Usage Access" — لازم تفعّل صلاحية Simple Launcher يدويًا من هناك (أندرويد ما يسمح بمنحها من نافذة الأذونات العادية).
6. عشان يصير هو اللانشر الافتراضي: اضغط زر Home بجهازك، اختر Simple Launcher، وحدد "Always".

## كيف تبنيه بدون لابتوب (GitHub Actions من الجوال)
1. سوّي حساب GitHub (لو ما عندك) وارفع محتوى هذا المجلد كـ repo جديد — تقدر تسوي هذا من متصفح الجوال أو تطبيق GitHub.
2. الملف `.github/workflows/build.yml` موجود جاهز بالمشروع — GitHub بيشغله تلقائيًا أول ما ترفع الكود.
3. روح لتبويب **Actions** بالـ repo، تابع الـ build (يأخذ 2-5 دقايق تقريبًا).
4. لما يخلص، روح لنفس الـ run، تحت **Artifacts** بتلقى ملف `SimpleLauncher-debug-apk` — نزّله على جوالك.
5. ثبّت الـ APK (لازم تفعّل "تثبيت من مصادر غير معروفة" بإعدادات الجوال أول مرة).
6. اتبع نفس خطوات 5-6 اللي فوق (Usage Access + تحديد اللانشر الافتراضي).

**ملاحظة:** هذا يبني نسخة Debug فقط (كافية للتجربة على جهازك الشخصي). لو حبيت تنشره على Google Play لاحقًا، بيحتاج توقيع (signing key) — خطوة إضافية نسويها وقتها.

## اللي موجود بهالنسخة (MVP)
- 8 صفحات (ViewPager2)، كل صفحة شبكة 4 أعمدة
- التطبيقات تترتب تلقائيًا من الأحدث استخدامًا للأقدم (قراءة عبر UsageStatsManager, آخر 30 يوم)
- كل ما ترجع للانشر يعيد الترتيب (onResume)
- الضغط على أيقونة يفتح التطبيق مباشرة
- مؤشر رقم الصفحة تحت (مثلاً "3 / 8")

## اللي ناقص عمدًا (مو ضمن MVP)
- تثبيت تطبيقات (Pin) بحيث ما تتحرك
- استبعاد تطبيقات من الترتيب التلقائي
- Widget أو بحث
- أيقونات مخصصة / تصميم

## أكبر خطر معروف
لو عدد التطبيقات المثبتة قليل (أقل من 8×4=32) بعض الصفحات بترجع فاضية — هذا سلوك متوقع، مو خطأ.
